`tree`是一个可以把目录以树形展示的工具，展示出来的目录相当的美观，如下：
```
├── Desktop
├── Documents
├── Downloads
├── Music
├── my
│   ├── create.sql.gz
│   ├── wordpress
│   ├── wordpress.gz
│   └── zabbix-release_3.4-1+xenial_all.deb
├── Pictures
├── Public
├── Templates
└── Videos
```

## 在ubuntu上安装

`sudo apt-get install tree`

## 一些用法

* `tree` 显示当前目录的树形结构
* `tree -L <level>`：指定目录深度
