更换配色的方法：
1. `mkdir ~/.vim/colors`
1. 把配色方案放到这下面
1. `.vimrc`中加入 `colorscheme  方案名`
```vim
syntax on
syntax enable
set t_Co=256
colorscheme molokai
```
