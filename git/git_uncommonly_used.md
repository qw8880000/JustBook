# 把其他分支的某个提交合并到当前分支

```sh
    git cherry-pick <commit id>
```

# log书写规范

第一行为标题，然后换行，写入详细内容，例如：
```
这是一个新功能:
* 如何实现
* 满足什么需求
* 功能如何使用
* 等等
```
